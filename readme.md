# はじめに

このソリューションはDDDでいうところのValueObjectにデータベースからデータを入れられないかと検証するためのサンプルプログラムになる。

本来であればデータベースからのデータ取得はないが、可用性を考えるとデータベースからの取得はありと考える。

# UnitVOSamle2 プロジェクト

このプロジェクトはUnityVOSample をもっと簡単に処理できないか検討したプロジェクトになる。

ValueObjectのBaseクラスで対処できないか参考までに記述したが、いろいろ問題があり、実行時エラーになる。

引数のないコンストラクタが必要なためになるが、結局引数なしのコンストラクタが必要であれば、UnitVOSampleが一番良い方法だと思う。
